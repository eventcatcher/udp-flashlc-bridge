# udp-flashlc-bridge

This repository aims to preserve and modernize the original udp-flashlc-bridge developed by Georg Kaindl, making it build and run on current versions of macOS while maintaining full compatibility with existing Flash applications.

Project Philosophy

This project is not a rewrite. It is a careful restoration of the original udp-flashlc-bridge, aiming to preserve its behavior while ensuring it remains buildable and usable on modern systems.
